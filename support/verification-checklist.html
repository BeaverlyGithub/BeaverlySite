Support Home Verification Checklist Contact and Escalation

UI location verification

  • Dashboard landing page label after login.
  • Flow navigation path and filter labels.
  • Instruction action labels (activate, pause, revoke).
  • Environment connection status labels.
  • Usage panel field names (percent used, used today, remaining).

Plan and billing verification

  • Current plan names and public descriptions.
  • Environment connection limits by tier.
  • Current pricing and cadence details.
  • Upgrade/downgrade timing behavior.
  • Refund and billing dispute handling policy.

Status and support workflow verification

  • Current status labels in production UI.
  • Expected lifecycle transitions for active instructions.
  • Which warnings appear in Flow for common failure cases.
  • Support contact mailbox ownership and response expectations.

Disclosure safety verification

  • No proprietary formulas or private decision internals disclosed.
  • No advisory language or guarantee phrasing.
  • Terminology remains user-facing: behavior, comfort, environment, instruction, Flow.
  • Non-custodial framing appears where relevant.
